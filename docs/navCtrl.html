ts.navCtrl(navCtrl, navParams, mongo, fb, alertCtrl, cookie)

Crea una instancia de AsociarPage.

Parameters

  1. navCtrl {NavController}

    contienelos controles de navegación.

  2. navParams {NavParams}

    contieneel parámetro que se le pasa a la vista cuando se la llama.

  3. mongo {MongoProvider}

    inicializala instancia del provider MongoProvider

  4. fb {FormBuilder}

    inicializauna instancia del builder FormBuilder

  5. alertCtrl {AlertController}

    inicializauna instancia del provider AlertController

  6. cookie {CookieService}

    inicializauna instancia del servicio CookieService

Inicializa las propiedades y comprueba si ya ha habido una conexión anterior revisando las cookies.

Podemos ver como en los parámetros que se le pasa hay objetos como "Provider","Builder" o "Service". Esta nomenclatura solo sirve para estructurar más el código. Realmente son todos Provider o Proveedores.
