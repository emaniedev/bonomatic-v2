AsociarPage.recuperarUser(data, err, funcion)

Esta es una función de utillaje. Se utiliza en nuestro constructor si se detecta que la asociación ya se ha hecho una vez. Recoge la información de la cookie y con ella encuentra el usuario. Si todo ha salido bien, pasa a su página de bonos. Utiliza nuestro proveedor MongoProvider.

Parameters

  1. data

    Este objeto contiene el usuario en cuestión, se pasa a json y se asigna

  2. err

    Si da error se ejecuta lo que este en este parametro

  3. funcion

    Cuando se completa la consulta a la bd se lanza la función que pongamos aqui.
